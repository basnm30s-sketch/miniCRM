The Glossary for Humans

API (Application Programming Interface)
The menu. It's the list of things you can ask the server to do. If it's not on the API, you can't order it.
CRUD
Acronym for Create, Read, Update, Delete. It's the four basic things you can do to data. Like "Write on paper, Read paper, Erase and Rewrite, Burn paper".
Component
A LEGO brick. A button, a form, a header. You build big things by snapping these small things together.
Endpoint
A specific URL where an API listens for orders. ex: /api/customers.
Full Stack
Someone who can cook (Backend) and serve tables (Frontend). You are becoming one.
JSON (JavaScript Object Notation)
The language data speaks. It looks like this: {"name": "Bond", "id": 007}.
State
What is happening right now. Is the user logged in? Is the modal open? State changes over time.
